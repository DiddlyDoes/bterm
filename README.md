# bterm

Fully customisable cross-platform terminal.

Runs everywhere. MacOS, Linux or Windows.

### Download

[http://bterm.bleenco.io](http://bterm.bleenco.io)

### Development

```sh
git clone https://github.com/bleenco/bterm.git
npm install
npm start
```

To make a production build run

```sh
npm run dist
```
